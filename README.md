# School Projects

A repository for collecting assignments and coursework completed during classes.

---

## 🗂️ Course Directory & Information

### [CSCI 1301] Introduction to Computing and Programming
* **Institution:** University of Georgia (UGA)
* **Language:** Java
* **Description:** Object-Oriented Programming (OOP) concepts. The project includes an interactive text-based fiction game.

### [STAT 4230] Applied Regression Analysis
* **Institution:** University of Georgia (UGA)
* **Language:** R
* **Description:** Covers diagnostics and model building for linear relationships.

### [STAT 4270] Network Data Analysis and Graphical Models
* **Institution:** University of Georgia (UGA)
* **Language:** Python
* **Description:** Includes a project on zip numbers classification using graphical modeling techniques.

### [STAT 4290] Nonparametric Methods
* **Institution:** University of Georgia (UGA)
* **Language:** Python
* **Description:** Includes a project on recognizing chemical substances without assuming underlying distributions.
